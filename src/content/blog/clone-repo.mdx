# Clonar repositorio de GitHub

Clonar un repositorio de GitHub es una tarea muy sencilla, pero es necesario tener instalado Git en tu computadora.
Si no lo tienes instalado, puedes descargarlo desde [aquí](https://git-scm.com/downloads).

Una vez que tengas instalado Git, puedes clonar un repositorio de GitHub de la siguiente manera:

1. Abre tu terminal.
2. Navega a la carpeta donde quieres clonar el repositorio.
3. Copia la URL del repositorio que quieres clonar. Puedes encontrarla en la página del repositorio en GitHub.
4. Escribe el siguiente comando en tu terminal, reemplazando `URL_DEL_REPOSITORIO` con la URL que copiaste:

```bash
git clone URL_DEL_REPOSITORIO
```

Por ejemplo, si quieres clonar el repositorio `soysergiogonza/prueba`:

```bash
git clone https://github.com/soysergiogonza/prueba.git
```

La URL también la puedes copiar desde el botón verde que dice "Code" en la página del repositorio. Al hacer clic en
el botón, se desplegará un menú con la URL del repositorio, puedes copiarla haciendo clic en el botón de copiar.

Una vez que hayas clonado el repositorio, tendrás una copia local del mismo en tu computadora. Puedes hacer cambios
en los archivos y subirlos de nuevo al repositorio en GitHub.
